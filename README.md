# Common Control

A new widget to replace the default widget


## How to use
A common control is a widget that replaces the default widget.



```dart
CColumn(
        padding: EdgeInset.all(10),
        margin: EdgeInset.all(10),
        onTab: () => {},
        flex: 1,
        gap: 10,
     ),
     
CRow(
        padding: EdgeInset.all(10),
        margin: EdgeInset.all(10),
        onTab: () => {},
        flex: 1,
        gap: 10,
     ),     
 ```
